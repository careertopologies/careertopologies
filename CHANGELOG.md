# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/), e este projeto adere ao [Versionamento Semântico](https://semver.org/lang/pt-BR/).

## [1.0.0]

### Adicionado
- Guia de implementação completo em português e inglês com roadmap detalhado
- Roadmap de implementação com timeline de 12 meses
- Tabelas detalhadas semana-a-semana para cada fase
- Alocação de recursos por papel e fase
- Itens do caminho crítico para cada fase

### Modificado
- Documentação de topologias (Y, W, Network) expandida em português e inglês
- Documentação de shapes (I, T, Pi) expandida em português e inglês
- Documentação de níveis de gestão (First-Line, Middle, Senior Leader) expandida em português e inglês
- README principal com links

### Removido
- Diagramas Mermaid substituídos por imagens PNG

## [0.1.0] - 2024

### Adicionado
- Framework inicial Career Topologies
- Documentação base em português e inglês
- Modelos de topologia: Y, W e Network
- Sistema de níveis baseado em Impacto × Autonomia
- Framework de competências
- Guias de referência para implementação
- Documentação de princípios e leis do framework
- Estrutura de remuneração e bandas salariais
- Licença Creative Commons Attribution 4.0

### Documentação
- README principal
- Guias em português (pt-BR)
- Guias em inglês (en-US)
- Referências e recursos adicionais
- Manifesto do projeto

## Tipos de Mudanças

- **Adicionado** - para novas funcionalidades
- **Modificado** - para mudanças em funcionalidades existentes
- **Descontinuado** - para funcionalidades que serão removidas
- **Removido** - para funcionalidades removidas
- **Corrigido** - para correções de bugs
- **Segurança** - para atualizações de segurança

---

**Nota**: As versões seguem o formato MAJOR.MINOR.PATCH onde:
- **MAJOR**: mudanças incompatíveis na API/estrutura
- **MINOR**: adição de funcionalidades compatíveis
- **PATCH**: correções de bugs compatíveis
